**Sede centrale:** Cortaccia (BZ) <br>

## Il nostro impegno

* Orario flessibile
* Mensa e bar interni
* Formazione continua 
* Un ambiente lavorativo dinamico, informale e collaborativo, ricco di proposte ed eventi aziendali 

## Se hai

* Una laurea in discipline STEM o equivalenti
* Almeno cinque anni di esperienza nello sviluppo software in contesti strutturati o su progetti complessi
* Solida conoscenza di almeno un linguaggio di programmazione di alto livello, come Java, C# o Go
* Esperienza nello sviluppo di applicazioni web e nell’integrazione di API REST o GraphQL
* Buona padronanza di HTML, CSS, JavaScript, TypeScript e familiarità con framework/librerie UI come React, Angular o Vue.js
* Ottima conoscenza di database relazionali (MSSQL, PostgreSQL, …) e attenzione all’ottimizzazione delle query e alla progettazione dei dati
* Esperienza con piattaforme cloud come Google Cloud Platform (GCP), Microsoft Azure o equivalenti, inclusi servizi di deploy, gestione container e integrazione CI/CD
* Esperienza con metodologie Agile/Scrum, strumenti di versioning (Git) e pratiche di CI/CD
* Spiccate capacità di problem solving e analisi architetturale
* Buona conoscenza della lingua italiana e inglese

## Se vuoi

* Contribuire allo sviluppo software secondo le esigenze dei vari reparti interni
* Definire l’infrastruttura e l’architettura più adatta per ogni progetto
* Occuparti della manutenzione e della documentazione delle soluzioni sviluppate
* Fornire supporto a livello globale per le soluzioni create
* Gestire progetti aziendali e contribuire attivamente alla loro realizzazione
