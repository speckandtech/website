## Il nostro impegno

* Orario flessibile
* Mensa e bar interni

## Se hai

* Una laurea in discipline STEM o equivalenti 
* Almeno cinque anni di esperienza professionale nello sviluppo front-end 
* Conoscenza approfondita di HTML, CSS, JavaScript, TypeScript e di framework/librerie UI come React, Angular o VueJs 
* Esperienza nello sviluppo con Node.js e nella configurazione di dev server come WebPack o Vite 
* Familiarità con API Rest e GraphQL 
* Spiccate capacità di problem solving e attenzione ai dettagli 
* Conoscenza della lingua italiana e inglese

## Se vuoi

* Contribuire allo sviluppo software secondo le esigenze dei vari reparti interni
* Definire l’infrastruttura e l’architettura più adatta per ogni progetto
* Occuparti della manutenzione e della documentazione delle soluzioni sviluppate
* Fornire supporto a livello globale per le soluzioni create
* Gestire progetti aziendali e contribuire attivamente alla loro realizzazione
